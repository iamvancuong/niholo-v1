# _ai-brain — Bộ Não AI của Niholo V1

Thư mục này là **kho lưu trữ trí tuệ** của dự án.
Mọi quyết định kiến trúc, kịch bản kỹ thuật, và tài liệu thiết kế đều được lưu ở đây.

---

## 📁 Danh sách tài liệu

| File | Mục đích |
|---|---|
| `00_implementation_plan.md` | Bản quy hoạch Kiến trúc tổng thể (Architecture, Tech Stack) & Schema Database cốt lõi. |
| `01_technical_screenplay.md` | Kịch bản chi tiết cho toàn bộ các luồng (Flow) từ Backend đến Frontend (Vue Components). |
| `02_data_strategy.md` | Chiến lược thu thập dữ liệu tự động, nguồn Audio (TTS/Hybrid), và cấu trúc Seeder. |
| `03_project_execution_phases.md` | Lộ trình thực thi dự án chia thành 10 Phase, gồm task Backend, Frontend, và Testing. |

---

## 🔖 Quy tắc đặt tên

```
00_  → Tài liệu chiến lược / kiến trúc tổng thể
01_  → Kịch bản kỹ thuật (BE + FE)
02_  → DB Migrations & Seeds (khi có)
03_  → Quyết định thay đổi thiết kế (ADR - Architecture Decision Records)
99_  → Ghi chú tạm / scratch
```
